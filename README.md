### 注意事项
该插件需要复写stop命令才能正常使用，使用前需要确定该命令未被其他插件注册  
否则可能会出现背包清空或背包回档的现象
### 使用方式  
重新加载插件配置文件
```
/NeotccInv reload
```
回档命令, 需要NeoTccInv.use权限
```
/NeotccInv rollback <玩家> <秒>
```
查看历史背包|末影箱
```
/NeotccInv history [inv|ender] <玩家> <秒>
```
查看玩家的背包
```
/NeotccInv see <玩家>
```
查看玩家的末影箱
```
/NeotccInv ender <玩家>
```
编辑玩家的背包|末影箱
```
/NeotccInv edit [inv|ender] <玩家>
```
